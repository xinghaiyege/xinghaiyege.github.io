  • FoamFix：https://www.curseforge.com/minecraft/mc-mods/foamfix-for-minecraft

    • 苹果皮（1.10.2+ 显示食物饱食度）：https://www.curseforge.com/minecraft/mc-mods/appleskin

  • JEI物品管理器（1.12+）：https://www.curseforge.com/minecraft/mc-mods/jei
  • NEI物品管理器（1.8+）：https://www.curseforge.com/minecraft/mc-mods/not-enough-items-1-8
  • NEI物品管理器（1.8-）：https://www.curseforge.com/minecraft/mc-mods/notenoughitems

  • 旅行者地图（Curse）：https://www.curseforge.com/minecraft/mc-mods/journeymap

  • 体素地图：https://www.curseforge.com/minecraft/mc-mods/voxelmap

  • Mowzie的怪物：https://www.curseforge.com/minecraft/mc-mods/mowzies-mobs
