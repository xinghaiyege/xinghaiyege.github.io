下载 Windows 10 系统

安装前，请确保 C 盘（系统盘）有 20 G 以上的存储空间。
下载前，请核对操作系统的位数。
下载前，请确保电脑中有 ED2K 下载软件（例如：迅雷）

» LTSC 版：

首推理由：系统占用存储少、内存占用少、CPU占用少。
无 Win10 应用商店、无微软小娜、无 UWP 应用，更新频率少而且稳定。
Win10 应用商店可通过手动安装，然后再安装其他的 UWP 应用。

  • 密钥：YJQNW-QH3Y8-JDHW2-MGV9F-2KCR9
  • 64位：ED2K 下载链接，发布时间 2019-03-15
  • 32位：ED2K 下载链接，发布时间 2019-03-15

» 企业版、教育版、专业版：

推荐怕给怕麻烦的人。
有 Win10 应用商店、有微软小娜、有 UWP 应用，更新频率频繁。
由安装密钥决定版本，激活后可通过密钥再次更换为别的版本再次激活。
